TableMorph 隱私權政策

最後更新日期：2026年1月23日

1. 簡介

TableMorph（以下簡稱「本擴充功能」）是由我們開發的 Chrome 瀏覽器擴充功能。 我們非常重視您的隱私權，本隱私權政策旨在說明我們如何收集、使用、儲存和保護您的資訊。 使用本擴充功能即表示您同意本隱私權政策的條款。

2. 資料收集

2.1 剪貼簿資料

本擴充功能需要存取您的剪貼簿內容，以執行表格格式轉換功能。 我們僅在您明確點擊轉換按鈕時才會讀取剪貼簿內容。

重要說明：

本擴充功能不會在背景自動讀取您的剪貼簿。 只有在您主動觸發轉換功能時，才會讀取剪貼簿中的表格資料。

2.2 資料類型

本擴充功能可能讀取的資料類型包括：

  • HTML 格式的表格資料（例如：從網頁或 Google Sheets 複製的內容）
  • 純文字格式的表格資料（例如：TSV、CSV 格式）
  • Markdown 格式的表格
  • LaTeX 格式的表格
  • JSON 格式的表格資料

3. 資料使用方式

我們使用您剪貼簿中的資料僅用於以下目的：

  • 格式轉換：將表格資料從一種格式轉換為另一種格式（Markdown、LaTeX、JSON、TSV）
  • 本地處理：所有轉換作業均在您的裝置本地進行
  • 寫回剪貼簿：將轉換後的結果寫回您的剪貼簿，方便您直接貼上使用
隱私承諾：

我們不會將您的資料用於任何其他目的，包括但不限於： 資料分析、廣告投放、資料銷售、或與第三方共享。

4. 資料儲存

本擴充功能不會儲存您的任何資料。 所有讀取的剪貼簿內容僅在記憶體中進行處理，轉換完成後立即清除。 我們不會將您的資料儲存在本地檔案、資料庫、或任何其他儲存媒介中。

5. 資料傳輸與共享

本擴充功能不會將您的資料傳輸到任何伺服器或第三方服務。 所有處理作業均在您的瀏覽器本地執行，不會有任何網路傳輸行為。

我們不會與任何第三方共享、出售、或租賃您的個人資料或剪貼簿內容。

6. 權限說明

本擴充功能需要以下權限才能正常運作：

  • clipboardRead：用於讀取剪貼簿中的表格資料
  • clipboardWrite：用於將轉換後的結果寫回剪貼簿

這些權限僅用於實現本擴充功能的核心功能，不會用於其他用途。

7. 第三方服務

本擴充功能使用以下開源套件進行資料處理，這些套件均在您的裝置本地執行：

  • PapaParse：用於解析 TSV/CSV 資料
  • Turndown：用於將 HTML 表格轉換為 Markdown
  • markdown-table：用於生成 Markdown 表格格式

這些套件不會將您的資料傳輸到外部伺服器，所有處理均在本地完成。

8. 資料安全

由於所有資料處理均在您的裝置本地進行，且不會儲存任何資料， 因此您的資料安全主要取決於您裝置的安全性。 我們建議您：

  • 保持您的瀏覽器和作業系統為最新版本
  • 使用強密碼保護您的裝置
  • 僅從可信來源安裝擴充功能

9. 您的權利

您對您的資料享有以下權利：

  • 存取權：由於我們不儲存資料，您可以直接查看剪貼簿內容
  • 控制權：您可以隨時選擇是否使用本擴充功能
  • 刪除權：您可以隨時解除安裝本擴充功能

10. 兒童隱私

本擴充功能並非專為 13 歲以下的兒童設計。 我們不會故意收集兒童的個人資訊。 如果您是家長或監護人，並發現您的孩子向我們提供了個人資訊， 請聯繫我們，我們將立即刪除該資訊。

11. 政策變更

我們可能會不定期更新本隱私權政策。 任何重大變更都會在擴充功能中或透過其他適當方式通知您。 建議您定期查看本頁面以了解最新的隱私權政策。
